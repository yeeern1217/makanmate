# Manual Test: 04 — Recommendation Card

## How to test

1. Open the app → go to the Radar map
2. Tap on a stall → tap CATCH
3. Watch the catch animation finish

- [ ] A "NEXT HIDDEN GEM" recommendation card slides up from the bottom
- [ ] It shows a stall name and heritage score
- [ ] Four score bars are visible: Content Match, Hidden Gem, Proximity, Diversity
- [ ] Small reasoning tags explain why this stall is recommended (e.g. "Hidden gem — only 12 reviews")
- [ ] After a moment, a friendly suggestion sentence appears (e.g. "Since you loved that wok-hei, hunt down the hidden Oh Chien stall nearby")

4. Tap "Navigate"

- [ ] Takes you to the Radar map with the recommended stall highlighted

5. Go back, catch another stall, and this time tap "Dismiss"

- [ ] Card disappears and you see the menu scan screen

6. Catch a few more stalls, then catch another one

- [ ] The recommendation changes based on what you've already caught (different stall suggested, different reasoning)

7. If you've caught all available stalls

- [ ] No recommendation card appears — goes straight to menu scan
